# shopping-system
Enjoy the pleasures of being a tycoon
使用說明:
A. 登入系統
    輸入帳號密碼後可選擇進入「前台購物」或「後台管理」,沒有帳密的話,亦可點擊「註冊」建立新的一般會員帳密並輸入暱稱。
________________________________________
B. 前台：客戶購物與收銀 (POS)
   進入前台後，模擬真實購物流程：
    1.	選購商品：點擊左側商品列表，輸入數量後加入購物車。
    2.	購物車管理：右側即時顯示購買明細與總金額。
    3.	結帳付款： 按下「結帳」後，支援多種擬真支付方式：
        ○	Apple Pay / LINE Pay：彈出模擬視窗，模擬手機掃描等待過程。
        ○	信用卡：彈出感應視窗，模擬連線銀行授權進度條。
        ○	現金支付：輸入支付的鈔票張數 (1000/500/100元)，系統自動計算找零。
        ○	賒帳：跳出警示視窗，提示將收取 日息 10% (10分利)，確認後才可成立訂單。
    4.	登出：點擊右上角「登出」可切換使用者。
________________________________________
C. 後台：管理員系統
    1. 商品管理 (CRUD)
        ●	瀏覽：列表顯示所有商品 ID、名稱、價格、庫存與描述。
        ●	自動填入：點擊下方表格任一行，資料自動帶入上方欄位。
        ●	編輯：可修改商品價格、補貨 (增加庫存) 或修改商品描述 (Description)。
        ●	刪除：具備防呆機制，需二次確認才可刪除商品。

2. 訂單管理 (財務報表)
        ●	訂單查詢：瀏覽所有歷史交易紀錄。
        ●	自動記錄：系統會自動抓取結帳當下的「會員名稱」寫入訂單，而非顯示訪客。
        ●	進階搜尋：可輸入「最低金額」與「最高金額」，篩選出特定價格範圍的訂單 (例如查詢高單價消費)。
